Email Notifications

Enable system email notifications. The email module will be wired up later; this flag controls whether it is active when it ships.

Center Deadline notification

Set the deadline by which center reps must complete program mapping. This date drives the center mapping-reminder emails.

Sending schedule
  • Emails go to each center's active center reps.
  • Weekly (Mondays) while the deadline is more than 3 days away, then daily in the final 3 days.
  • Sending stops automatically once a center reaches the 90% mapping target, or once the deadline date passes. Centers with no portfolio or no reps are skipped, and no one is emailed twice in the same day.
  • Requires this deadline to be enabled with a date set. Actual delivery is gated by the Email Notifications toggle above.

Programs Deadline notification

Set the deadline by which program reps must respond to their mappings. This date drives the program mapping-reminder emails.

Sending schedule
  • Emails go to active program reps of programs that still have mappings awaiting a response.
  • Daily up to and including the deadline day, then sending stops.
  • Programs with nothing pending or no reps are skipped, and no one is emailed twice in the same day.
  • Requires this deadline to be enabled with a date set. Actual delivery is gated by the Email Notifications toggle above.
